# TCGA-LUAD-Prognostic-Biomarker-Pipeline
A data wrangling and downstream analysis pipeline for TCGA-LUAD. The project aims to build a prognostic model for patient survival. Workflow: tumor vs normal DGE analysis, Univariate Cox, Elastic Net regression, model training(Multivariate Cox Regression / Random Forest Survival), evaluation(C-index / Time-dependent ROC-AUC).
